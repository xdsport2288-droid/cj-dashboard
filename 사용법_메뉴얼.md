# 🚀 CJ 대시보드 자동화 시스템 사용 매뉴얼

이 시스템은 엑셀 파일을 저장하기만 하면, 자동으로 데이터를 변환하여 웹 대시보드에 배포해 주는 자동화 프로그램입니다.

## 1. 감지기 켜기 (이제 자동으로 켜집니다!)
원래는 컴퓨터를 켤 때마다 한 번씩 실행해야 했으나, **윈도우 시작프로그램에 자동 등록**해 두었습니다!
따라서 앞으로는 컴퓨터를 껐다 켜도 아무것도 누르실 필요가 없습니다. (부팅 시 투명 모드로 알아서 켜집니다.)

*혹시라도 감지기를 강제로 껐다가 다시 켜야 할 경우에만 이 폴더의 `자동감지시작.bat`을 더블클릭하시면 됩니다.*

## 2. 평상시 업무 (매일 하는 작업)
감지기가 켜져 있다면 이제 평소처럼 엑셀만 수정하시면 됩니다.

1. 같은 폴더에 있는 **`위니온.xlsx`** 파일을 엽니다. (다른 이름의 엑셀이나 csv 파일을 덮어씌워도 됩니다.)
2. 데이터를 평소처럼 수정합니다.
3. **저장(Ctrl+S)** 버튼을 누릅니다.
4. 약 1~3초 뒤 화면 최상단에 **"배포 성공!"** 이라는 팝업 알림이 뜹니다.
5. 스마트폰이나 PC로 대시보드 웹사이트에 들어가 보시면 방금 수정한 데이터가 반영되어 있습니다!

## 💡 참고 사항
- 엑셀 파일을 열어둔 채로 수정하고 계속 저장을 누르셔도 전혀 문제없이 계속 감지되고 자동 배포됩니다.
- 감지기를 끄고 싶으시다면 컴퓨터를 껐다 켜시거나, 작업 관리자에서 `python.exe` 프로세스를 종료하시면 됩니다.
